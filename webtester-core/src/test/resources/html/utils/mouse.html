Mouse Test Page

This page contains elements for testing the functionality of the Mouse utility class.

The first scenario is used to demonstrate how to 'move' the mouse over one or several elements of an page with various delays between the actions.
There are three buttons:
  1. The first button 'Start Button' is initially displayed
  2. The second button 'Instant Button' is displayed as soon as the mouse is moved over the 'Start Button'.
  3. The third button 'Delayed Button' is displayed after the mouse is moved over the 'Instant Button' with a delay of 500ms.








The second scenario is used to demonstrate how to double click an element.
There are three buttons:
  1. The first button 'Single Click Button' is initially displayed
  2. The second button 'Double Click Button' is initially displayed
  3. The third button 'Target Button' is displayed after the 'Single Click Button' was single clicked or the 'Double Click Button' was double clicked.
